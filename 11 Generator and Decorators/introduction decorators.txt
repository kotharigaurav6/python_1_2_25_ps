What is a Decorator?

A decorator is a function that modifies another function’s behavior without changing its actual code.
In other words “Wrapping a function with extra functionality”

🔹 Basic Idea (Step by Step)
1. Functions are objects in Python
def greet():
    print("Hello")

x = greet
x()   # Hello

2. Function inside function
def outer():
    def inner():
        print("Inside inner")
    inner()

outer()

3. Returning a function
def outer():
    def inner():
        print("Hello from inner")
    return inner

func = outer()
func()

Now combine everything → Decorator

def my_decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

🔹 Using decorator manually

def say_hello():
    print("Hello")

say_hello = my_decorator(say_hello)

say_hello()


Output:

Before function
Hello
After function

🔹 Using @ syntax (real decorator style)
@my_decorator
def say_hello():
    print("Hello")

say_hello()


Same output, cleaner syntax
🔹 Decorator with Arguments
def my_decorator(func):
    def wrapper(name):
        print("Before")
        func(name)
        print("After")
    return wrapper

@my_decorator
def greet(name):
    print("Hello", name)

greet("Gaurav")

Simple One-Line Understanding
Decorator = function that takes a function, modifies it, and returns a new function
